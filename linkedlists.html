Linked Lists

What is a Linked List?

A Linked List is a linear data structure where each element (called a node) contains:

  • Data
  • A pointer (or reference) to the next node in the sequence

Types of Linked Lists:

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List (Singly or Doubly)

Singly Linked List

Structure:

Each node contains two parts: the data and a reference to the next node in the list.

Visual

[Data | Next] -> [Data | Next] -> [Data | Next] -> NULL

Concepts:

  • Node: A container that stores data and the next reference.
  • Head: Starting point of the list.
  • Traverse: Move through each node.

Step 1: Define a Node

Python

class Node:
    def __init__(self, data):
        self.data = data       # store the value (like "Apple")
        self.next = None       # points to the next node (default is None)

Weâ€™re creating a blueprint called Node.

Each node holds some data (in our case, a fruit name).

.next is a pointer to the next node in the list.

Step 2: Create Nodes

Here we're creating 3 nodes, each with a fruit name as its data.

Step 3: Link the Nodes

Python

We make n1 point to n2, so now Apple â†’ Banana.

Step 4: Traverse the List

    print(temp.data, end=" -> ")
    temp = temp.next

temp = n1: Start from the head (first node).

temp = temp.next: Move to the next node.

After the loop, we print "None" to show the list ends.

Summary

A linked list is just a bunch of Node objects connected together.
