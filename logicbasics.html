Boolean Algebra

Boolean Algebra is a branch of mathematics that deals with variables having two distinct values: 0 and 1. It is the foundation of digital logic and circuit design.

\( (x' + y' + z') (x' + y' + z) \)

\( (x + y' + z) (x + y + z') = F_1' \)

Sum of Minterms and Product of Maxterms

The Sum of Minterms (SOM) is a canonical form where a Boolean function is expressed as the OR (sum) of all minterms that produce a value of 1.

The Product of Maxterms (POM) is a canonical form where a Boolean function is expressed as the AND (product) of all maxterms that produce a value of 0.

These standard forms are useful for simplifying and analyzing Boolean functions in digital logic design.

Sum of Minterms Example

The Boolean function \( F = A + B'C \) can be expressed as a sum of minterms. This function has three variables: \( A \), \( B \), and \( C \).

Since the first term \( A \) is missing two variables, we expand it:

\[ A = A(B + B') = AB + AB' \]

This function still lacks one variable:

\[ A = AB(C + C') + AB'(C + C') \\ = ABC + ABC' + AB'C + AB'C' \]

The second term \( B'C \) is missing one variable:

\[ B'C = B'C(A + A') = AB'C + A'B'C \]

Combining all terms:

\[ F = A + B'C \\ = ABC + ABC' + AB'C + AB'C' + A'B'C \]

Since \( AB'C \) is repeated, according to the theorem \( x + x = x \), we can remove one of them. Ordering the minterms:

\[ F = A'B'C + AB'C' + AB'C + ABC' + ABC \\ = m_1 + m_4 + m_5 + m_6 + m_7 \]

Sometimes, it is better to express the Boolean function as a sum of minterms in the following concise form:

\[ F(A, B, C) = \Sigma(1, 4, 5, 6, 7) \]

Product of Maxterms Example

Express the Boolean function \( F = xy + x'z \) as a product of maxterms. First, rewrite the function using the distributive law into OR terms:

\[ F = xy + x'z = (xy + x')(xy + z) \\ = (x + x')(y + x')(x + z)(y + z) \\ = (x' + y)(x + z)(y + z) \]

The function has three variables: \( x \), \( y \), and \( z \). Each OR term is missing one variable, so we expand them:

\[ x' + y = (x' + y + z)(x' + y + z') \\ x + z = (x + y + z)(x + y' + z) \\ y + z = (y + x + z)(y + x' + z) \]

Combining all terms and removing duplicates, we get:

\[ F = (x + y + z)(x + y' + z)(x' + y + z)(x' + y + z') \\ = M_0 M_2 M_4 M_5 \]

The simplified representation is:

\[ F(x, y, z) = \prod (0, 2, 4, 5) \]

The symbol \( \prod \) indicates the AND of maxterms. The numbers in parentheses represent the maxterm indices.
