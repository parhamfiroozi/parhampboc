Vector Spaces

Okay, so the first topic: vector spaces. Doesn’t need an explanation — you already know what it is. I try to learn it through examples.

A vector space is a set of objects (called vectors) that can be added together and multiplied by scalars, with the result still belonging to the set. These operations must satisfy eight specific rules, such as associativity and distributivity.

Subspace

A subspace is a subset of a vector space that is itself a vector space. It must be closed under vector addition and scalar multiplication.

Example

Let V = ℝ³. Define the set:

W = { (x, y, z) ∈ ℝ³ | x + y + z = 0 }

  • Closed under addition: If x₁ + y₁ + z₁ = 0 and x₂ + y₂ + z₂ = 0, then:
    (x₁ + x₂) + (y₁ + y₂) + (z₁ + z₂) = 0.
  • Closed under scalar multiplication: For scalar a, a(x + y + z) = a ⋅ 0 = 0.

Therefore, W is a subspace of ℝ³.

Span

Example in ℝ³

Let v₁ = (1, 1, 0), v₂ = (0, 1, 1).

Take a linear combination:

a ⋅ v₁ + b ⋅ v₂ = a ⋅ (1, 1, 0) + b ⋅ (0, 1, 1)

  • First component: a
  • Second component: a + b
  • Third component: b

Result: (a, a + b, b)

Example in ℝ⁴

Let:

  • v₁ = (1, 2, 0, 1)
  • v₂ = (0, 1, 1, 1)
  • v₃ = (1, 3, 1, 2)

Check if v₃ ∈ span{v₁, v₂}

Set: a ⋅ v₁ + b ⋅ v₂ = v₃

Left side: (a, 2a + b, b, a + b) = (1, 3, 1, 2)

  • a = 1
  • b = 1

So v₃ ∈ span{v₁, v₂}

Span means all combinations of given vectors. You're asking: "Can I build this using only those?" If yes, it's in the span.

Linear Independence

Imagine you have some sticks. If no stick can be made by mixing the others, they are independent. But if you can make one stick by using the others, they are dependent.

Example

Given:

  • v₁ = (1, 2, 3, 4)
  • v₂ = (2, 4, 6, 8)
  • v₃ = (1, 0, 1, 0)

Solve: a ⋅ v₁ + b ⋅ v₂ + c ⋅ v₃ = 0

Leads to system:

  • a + 2b + c = 0
  • 2a + 4b = 0
  • 3a + 6b + c = 0
  • 4a + 8b = 0

Solving gives: a = -2b, b = free, c = 0 ⇒ nontrivial solution exists ⇒ linearly dependent.

Basis

A basis is like a set of special building blocks. You only need a few of them to build everything in your space.

They must satisfy two conditions:

  • No vector in the set can be written as a combination of the others (they are linearly independent).
  • The set can be used to generate every vector in the space (they span the space).

Example in ℝ²

Basis: { (1, 0), (0, 1) }

Determine Basis

Check if B = { (1, 0), (0, 1) } is a basis of ℝ²:

Step 1: Linear Independence

Solve: a(1, 0) + b(0, 1) = (0, 0) ⇒ a = 0, b = 0 ⇒ independent.

Step 2: Spanning

Any (x, y) ∈ ℝ² can be written as x(1, 0) + y(0, 1) ⇒ spans ℝ².

Step 3: Conclusion

Linearly independent and spans ℝ² ⇒ it is a basis.

Dimension

The dimension tells you how many basis vectors (or "special blocks") are needed to describe the entire space.

Examples:

  • ℝ² (a flat table world): needs 2 blocks — forward and sideways
  • ℝ³ (full 3D space): needs 3 blocks — forward, sideways, and up

So:

  • Flat world ⇒ dimension = 2
  • Space world ⇒ dimension = 3
